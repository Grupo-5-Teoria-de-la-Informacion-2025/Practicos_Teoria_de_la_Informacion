Algoritmo de Factorización de Fermat

Este script implementa el Algoritmo de Factorización de Fermat, 
una herramienta utilizada en criptoanálisis para romper el criptosistema RSA.
Su función es encontrar los factores primos p y q de un módulo N ingresado por el usuario, 
siempre que los factores sean cercanos entre sí.


 Librerías Utilizadas

math: Para realizar cálculos matemáticos esenciales como la raíz cuadrada entera (`isqrt`).
time: Se usa para medir el tiempo de ejecución y establecer un límite de duración para el algoritmo.

Instalación


math y time: No requiere. Se instalan al instalar Python (son parte de la biblioteca estándar)
